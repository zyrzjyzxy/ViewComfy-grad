# 需求再次确定

管理员功能中需要实现的是：
- 用户管理：包括创建、删除、更新用户信息，以及重置用户密码。
- 生成记录管理：包括查看所有用户的生成记录，以及删除指定记录或批量删除记录。
- 统计数据：包括查看系统的用户数量、生成记录数量等统计信息。

以上三个功能应该分别对应后端的 API 接口，前端界面则通过调用这些接口来实现管理员功能。
在登录管理员身份后，上述三个功能应该在侧边栏显示对应的入口，用户点击后可以跳转到相应的页面。
管理员工作台与用户工作台的界面应该有区分，而不是显示在同一个页面。
登录管理员身份后，应该要跳转到管理员工作台页面 /admin/page.tsx。

其中，管理员工作台主页是http://localhost:3000/admin（包含统计数据）
管理员用户管理界面是http://localhost:3000/admin/users
管理员生成记录管理界面是http://localhost:3000/admin/histories
