## 登录界面修改计划

### 修改目标
1. **调整布局**：使图片居于左侧，登录表单居于右侧
2. **增加返回按钮**：添加"返回首页"按钮
3. **更新图片**：使用时尚服装设计图替换现有图片
4. **优化样式**：调整整体布局以符合参考设计

### 具体修改步骤

1. **修改登录页面布局结构**
   - 调整 `lg:grid-cols-2` 布局，确保图片在左侧，表单在右侧
   - 为左侧图片区域添加适当的样式和布局

2. **添加返回首页按钮**
   - 在登录表单区域顶部添加"返回首页"按钮
   - 按钮链接到根路径 "/"
   - 添加适当的图标和样式

3. **更新左侧图片**
   - 将左侧图片替换为时尚服装设计图
   - 确保图片正确显示和响应式调整

4. **优化表单样式**
   - 调整表单标题为"登录"
   - 优化表单元素的布局和样式
   - 确保整体设计与参考图一致

### 修改文件
- `app/login/[[...login]]/page.tsx` - 主要修改文件

### 技术实现
- 使用 Tailwind CSS 进行布局和样式调整
- 使用 Next.js 的 Image 组件加载图片
- 确保响应式设计在不同屏幕尺寸下正常显示
